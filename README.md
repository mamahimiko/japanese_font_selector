# Japanese Font Explorer　”Moji-Lab”

A React Single Page Application that allows users to explore and preview Japanese fonts interactively.

## Overview

This project was built as part of a React assignment focusing on core concepts such as state, props, and component-based architecture.

Users can:
- Preview different Japanese fonts in real time
- Filter fonts by use case (e.g. web, poster, branding)
- Explore font styles through a clean and modern UI

## Tech Stack

- React
- JavaScript (ES6+)
- CSS Modules
- Material UI (MUI)
